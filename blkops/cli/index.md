# blkops CLI 使用说明

blkops 是一个命令行工具，所有功能均通过参数控制。

## 全局帮助
```bash
blkops --help
```

## 命令速览

| 命令 | 说明 |
|---|---|
| `-s, --search` | 搜索分区 |
| `-w, --write` | 写入镜像 |
| `-d` | 导出分区 |
| `-p` | 仅输出设备路径 |
| `-h` | 显示帮助 |
