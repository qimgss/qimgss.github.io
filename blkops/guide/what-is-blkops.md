# 什么是 blkops

**blkops** 是一个**纯 Rust 编写的 Android CLI 工具**，用于在 Android 设备上：

- 查找 block 分区
- 向分区写入镜像
- 从分区导出镜像

## 为什么选择 blkops

| 特性 | 说明 |
|---|---|
| 🦀 纯 Rust | 无 Java / Python 运行时依赖 |
| ⚡ 高性能 | 直接操作 block 设备 |
| 🛠 跨平台 | 支持 Windows / Linux 构建 |
| 📱 Android 专注 | 针对 Android 分区设计 |
