# blkops

一个 **纯 Rust** 编写的 Android CLI 工具，用于查找和刷写块设备分区。

![Rust](https://img.shields.io/badge/rust-1.70%2B-orange.svg)
![License](https://img.shields.io/badge/License-Apache--2.0-blue.svg)

## 功能特性

- 🔍 搜索 Android 设备的 block 分区
- 💾 写入镜像到指定分区
- 📤 导出分区为镜像文件
